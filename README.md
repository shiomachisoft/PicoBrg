# PicoBrg
## 1. 概要  
- (1)マイコン基板はRaspberry Pi Pico Wを使用します。
- (2)BLE⇔UART、Wi-Fi(TCP ソケット通信)⇔UART  の変換(ブリッジ)を行います。
- (3)Pico Wに対するWi-Fi設定・UART設定は専用PCアプリを使用し、Pico WのFlashメモリに保存します。
